## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Pull or push

git status
git add -A
git commit -m ''
git pull origin main


git status
git add -A
git commit -m ''
git status
git push origin HEAD
